Description

Snap Puzzle is a simple game I created for an activity we had in our AI class. I chopped Katya Lischina's photo into nine equal parts. It is basically just the same with slide puzzle. The only difference is the pieces basically snap instead of slide. The reason is that we used a C# Windows Forms Application. Adding animation would not be efficient for this simple activity we had for an hour.
